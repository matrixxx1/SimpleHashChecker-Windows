# m3Coding: Simple Hash Checker

## Short Title

m3Coding: Simple Hash Checker

## Short Description

Calculate MD5, SHA1, SHA256, and SHA512 hashes for files and compare them with an expected value.

## Full Description

m3Coding: Simple Hash Checker is a focused Windows utility for verifying file integrity. Drop a file into the app or choose one with the browse button, then calculate MD5, SHA1, SHA256, or SHA512 from one clean screen.

The app shows file details, places the calculated digest in a selectable textbox, and includes a copy-to-clipboard button for fast sharing. Paste an expected hash to immediately see whether the calculated value matches.

Everything runs locally on your Windows device. No account is required, and files are read only for hash calculation.

## Feature Bullets

- Drag and drop files into the app
- Browse for files with the standard Windows file picker
- Calculate MD5, SHA1, SHA256, and SHA512 hashes
- Copy calculated hashes to the clipboard
- Compare against an expected hash value
- Runs locally with no account requirement
